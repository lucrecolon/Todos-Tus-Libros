# 📚 Todos Tus Libros - Plataforma de Búsqueda

## 📖 Sobre el Proyecto

"Todos Tus Libros" es una web diseñada para centralizar y facilitar la búsqueda de libros en múltiples librerías de Argentina. La plataforma permite a los usuarios descubrir nuevas lecturas, consultar precios y disponibilidad de stock en distintas sucursales, y gestionar sus selecciones de forma intuitiva.

## ✨ Funcionalidades Principales

* **Búsqueda Avanzada:** Motor de búsqueda integrado para filtrar publicaciones por título, autor o editorial.
* **Geolocalización y Disponibilidad:** Cálculo de distancias en tiempo real para mostrar en qué librerías cercanas hay stock de un libro específico.
* **Perfil de Usuario:** Panel de autogestión donde el usuario puede editar sus datos personales y mantener actualizada su información de contacto.
* **Carrito de Compras:** Panel de gestión de libros seleccionados con cálculo automático de totales y preparación para el flujo de checkout.
* **Lista de Favoritos (Wishlist):** Sección dedicada para guardar y administrar los títulos de interés para futuras compras.
* **Vidriera de Novedades:** Página de inicio dinámica que expone los últimos lanzamientos y recomendaciones.

## 🛠️ Stack Tecnológico

**Frontend:**
* **Core:** React 19 + TypeScript
* **Entorno & Compilación:** Vite
* **Enrutamiento:** React Router DOM v7
* **Estilos:** CSS Vanilla (Diseño Responsive adaptado a Mobile First)
