# 📚 Todos Tus Libros - Plataforma de Búsqueda

## 📖 Sobre el Proyecto
"Todos Tus Libros" es una web diseñada para centralizar y facilitar la búsqueda de libros en múltiples librerías de Argentina. La plataforma permite a los usuarios descubrir nuevas lecturas, consultar precios y disponibilidad de stock en distintas sucursales, y gestionar sus selecciones de forma intuitiva.

Este proyecto actúa como el escaparate digital de la red de librerías usuarias de UltraGestión, brindando una experiencia de usuario fluida, rápida y moderna.

## ✨ Funcionalidades Principales
* **Búsqueda Avanzada:** Motor de búsqueda integrado para filtrar publicaciones por título, autor o editorial.
* **Vidriera de Novedades:** Página de inicio dinámica que expone los últimos lanzamientos y recomendaciones del mes.
* **Carrito de Compras:** Panel de gestión de libros seleccionados con cálculo automático de totales.
* **Lista de Favoritos (Wishlist):** Sección dedicada para que el usuario pueda guardar y administrar los títulos que le interesan para futuras compras.

## 🛠️ Stack Tecnológico
* **Core:** React 19 + TypeScript
* **Entorno & Compilación:** Vite
* **Enrutamiento:** React Router DOM v7
* **Estilos:** CSS
* **Gestión de Estado:** React Context API + LocalStorage Browser API
